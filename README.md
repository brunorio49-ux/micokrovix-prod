
# Micokrovix

Ferramenta para caminhoneiros calcularem o lucro real de fretes.

## Funcionalidades atuais

- Calculadora de frete
- Cálculo de custo de diesel
- Cálculo de pedágio
- Estimativa de lucro

## Tecnologias usadas

- Python
- Flask
- SQLite
- HTML

## Objetivo do projeto

Criar uma ferramenta simples e eficiente para ajudar caminhoneiros a saber se um frete vale a pena antes de aceitar.

Projeto em desenvolvimento.
